# 1. ORAL SOLID DOSAGE FORMS (OSD)

  

> OSD forms account for **~70% of all pharmaceutical products** and are the #1 focus area for Indian generics companies. This is the bread and butter of formulation R&D.

## 1.1 TABLETS — Immediate Release (IR)

  

### A) Complete Theory

  

**IR tablets** are designed to release the drug rapidly after administration, typically achieving **>85% dissolution within 30 minutes** (USP). The drug disintegrates from the tablet → dissolves in GI fluids → absorbs through GI membrane.

  

**Key Concepts:**

- **Compression physics:** Plastic deformation (MCC) vs brittle fracture (lactose, DCP) — bonding mechanisms determine tablet strength

- **Powder flow:** Carr's Index (<15% = excellent), Hausner Ratio (<1.18 = good), Angle of Repose (<30° = excellent)

- **Compactibility:** Tabletability (tensile strength vs compaction pressure), compressibility (porosity vs pressure), compactability (tensile strength vs porosity)

- **Wetting & disintegration:** Capillary action pulls water into tablet → disintegrant swells → tablet breaks → drug dissolves

  

### B) Critical Formulation Parameters

| Parameter | Target | Why It Matters |
|:---|:---|:---|
| **Blend uniformity** | RSD ≤5% (USP <905>) | Ensures each tablet has correct dose |
| **Granule flow** | Carr's Index <20% | Consistent die filling → consistent weight |
| **Granule moisture** | 1–3% (typical) | Too wet = sticking; too dry = capping |
| **Compression force** | Formulation-dependent | Too high = capping/lamination; too low = friable |
| **Hardness** | 4–8 kp (typical) | Must survive handling but still disintegrate |
| **Disintegration time** | <15 min (USP) for IR | Regulatory requirement |
| **Dissolution** | Q ≥80% in 30 min (typical) | Bioavailability assurance |
| **Friability** | NMT 1.0% w/w (USP) | Tablet integrity during shipping |
  

### C) Key Excipients and Roles

| Category | Excipient | Concentration | Role |
|:---|:---|:---|:---|
| **Diluent/Filler** | Microcrystalline Cellulose (MCC, Avicel PH-102) | 20–90% | Compressibility, flow (DC grade) |
| **Diluent** | Lactose monohydrate (FastFlo 316) | 15–75% | Filler, good flow (spray-dried grade for DC) |
| **Diluent** | Dibasic calcium phosphate (DCP, Emcompress) | 20–50% | Hard, brittle filler |
| **Binder (dry)** | PVP K30 (Povidone) | 2–5% (dry), 5–10% in solution | Granule strength |
| **Binder** | HPMC E5 (Methocel) | 2–5% | Wet granulation binder |
| **Disintegrant** | Croscarmellose sodium (Ac-Di-Sol) | 2–5% (intragranular + extragranular) | Wicking + swelling mechanism |
| **Disintegrant** | Sodium starch glycolate (Explotab) | 2–8% | Swelling (200–300x volume) |
| **Disintegrant** | Crospovidone (Polyplasdone XL) | 2–5% | Wicking + elastic recovery |
| **Lubricant** | Magnesium stearate | 0.25–1.0% | Reduces die-wall friction; **mix only 2–5 min** |
| **Glidant** | Colloidal silicon dioxide (Aerosil 200) | 0.1–0.5% | Improves powder flow |
| **Surfactant** | Sodium lauryl sulfate (SLS) | 0.5–2.0% | Wetting agent for hydrophobic APIs |
  

### D) Manufacturing Process — Direct Compression (DC)

  

```

API Sieving (#30-40 mesh)

       ↓

Excipient Sieving (#30 mesh)

       ↓

Pre-blending (API + diluent + disintegrant) → V-blender or Bin blender (10-15 min)

       ↓

Add glidant (Aerosil) → Blend 3-5 min

       ↓

Add lubricant (Mg stearate) → Blend 2-5 min ONLY (over-mixing reduces hardness)

       ↓

Compression → Rotary tablet press (e.g., Cadmach, Fette)

       ↓

In-process controls: Weight, Hardness, Thickness, Disintegration

       ↓

Finished tablet → QC testing

```

  

**Wet Granulation Process:**

  

```

API + Diluent + Disintegrant (intragranular) → Dry mix in RMG (2-5 min)

       ↓

Add binder solution (PVP in water/IPA) → Granulate in RMG (3-10 min)

       ↓

Wet screen (#8-12 mesh)

       ↓

Drying → FBD (Fluid Bed Dryer) at 50-60°C to LOD 1-3%

       ↓

Dry screen (#20-30 mesh) → Sizing

       ↓

Add extragranular disintegrant + glidant → Blend 10-15 min

       ↓

Add lubricant → Blend 2-5 min

       ↓

Compression → Rotary tablet press

```

| Test | Method | Acceptance Criteria |
|:---|:---|:---|

| **Friability** | USP <1216> | NMT 1.0% (20 tablets, 100 revolutions, 4 min) |
| **Disintegration** | USP <701> | NMT 30 min (uncoated); NMT 15 min (typical target) |
| **Dissolution** | USP <711> | Q ≥80% in 30 min (or as per monograph) |
| **Related Substances** | HPLC | Per ICH Q3B: individual ≤0.2%, total ≤1.0% (typical) |
| **Moisture** | Karl Fischer / LOD | Per specification (typically NMT 2–3%) |
| **Microbial Limits** | USP <61>/<62> | TAMC ≤10³ CFU/g; TYMC ≤10² CFU/g; absent E. coli |



### F) Common Problems and Troubleshooting

  

| Problem                    | Root Cause(s)                                                      | Solution(s)                                                                   |
| :------------------------- | :----------------------------------------------------------------- | :---------------------------------------------------------------------------- |
| **Capping**                | Air entrapment; insufficient binder; excessive fines; low moisture | Increase binder; add pre-compression; reduce speed; optimize LOD              |
| **Lamination**             | Excessive compression force; air entrapment; oily/waxy API         | Reduce force; add pre-compression; modify granulation                         |
| **Sticking/Picking**       | Excess moisture; low melting point API; punch surface defects      | Reduce moisture; add anti-adherent (talc); polish punches; add colloidal SiO₂ |
| **Weight Variation**       | Poor flow; inconsistent die fill; segregation                      | Improve flow (glidant); optimize particle size; use force-feeder              |
| **Low Hardness**           | Insufficient binder; over-lubrication; excess moisture             | Increase binder; reduce Mg stearate mixing time; optimize LOD                 |
| **High Friability**        | Insufficient bonding; excess fines                                 | Increase binder; increase compression force; add MCC                          |
| **Slow Dissolution**       | Over-lubrication; hydrophobic API; excess hardness                 | Reduce Mg stearate; add SLS; increase disintegrant; reduce compression        |
| **Content Non-uniformity** | Segregation; poor mixing; cohesive API                             | Improve blending; use ordered mixing; granulate the API                       |
| **Mottling**               | Dye migration; uneven drug distribution                            | Use lake dyes (not soluble dyes); improve binder distribution                 |

### H) Top 10 Interview Questions — IR Tablets

  

**Q1: "What factors do you consider when choosing between direct compression and wet granulation?"**

> **Model Answer:** I evaluate: (1) API properties — if API has good flow and compressibility (e.g., Metformin HCl at high dose), DC may be possible with DC-grade excipients. If API is cohesive, hygroscopic, or low-dose (<1mg), WG is preferred for content uniformity. (2) Drug-excipient compatibility — moisture-sensitive APIs (e.g., aspirin) favor dry granulation/DC. (3) Dose — high-dose drugs often need WG to achieve acceptable tablet size. (4) Cost — DC has fewer steps, lower cost. (5) Scale-up complexity. In my thesis, I chose WG because my API had poor flow (Carr's Index 32%) and dose was 200mg.

  

**Q2: "Why is magnesium stearate mixed for only 2–5 minutes?"**

> Over-mixing causes Mg stearate to form a hydrophobic film over granules, reducing interparticle bonding (lower hardness), increasing disintegration time, and decreasing dissolution. Typical mixing is 2–5 min at 12–15 RPM in a V-blender. Over-lubrication is a common cause of dissolution failure at scale-up. I always time lubricant mixing precisely and monitor hardness/dissolution as controls.

  

**Q3: "Your tablet has capping issues at production scale. Walk me through your troubleshooting."**

> Systematic approach: (1) Check granule moisture (LOD) — if <1%, granules are too dry and brittle → adjust drying endpoint. (2) Check compression parameters — reduce main compression force; increase pre-compression force to deaerate. (3) Check granule PSD — excess fines (<150 µm) cause poor rearrangement → optimize screening/milling. (4) Evaluate binder concentration — increase by 10–20% and run trial. (5) Check punch condition — worn punches cause localized stress. (6) If persistent, consider changing granulation method (e.g., from dry to wet granulation for better bonding). I would run a small-scale DoE varying moisture, pre-compression force, and binder level.

  

**Q4: "Explain the difference between super-disintegrants. When would you choose each?"**

> **Croscarmellose sodium (Ac-Di-Sol):** Wicking + swelling. Best all-around; 2–5%. Works well in DC and WG. Effective at low concentrations. **SSG (Explotab):** Extreme swelling (300x). Works well but can affect dissolution if used >8% (gel formation). Best for WG. **Crospovidone (Polyplasdone XL):** Wicking + elastic recovery (returns to original shape). Non-swelling, so doesn't form gel — best for high-dose APIs where gel from SSG might retard dissolution. Choice depends on: (1) API compatibility with moisture (SSG retains water), (2) tablet size constraints, (3) drug release requirements.

  

**Q5: "How do you ensure blend uniformity for a low-dose drug (e.g., 0.5 mg)?**

> Challenges: Segregation, poor distribution. Solutions: (1) **Geometric dilution** — premix API with a portion of MCC by doubling. (2) **Ordered mixing** — use fine API particles that adhere to carrier surface (MCC). (3) **Wet granulation** — dissolve API in binder solution to ensure molecular-level distribution. (4) **Use smaller blender fill** (50–70%) for better mixing. (5) **Validate** using stratified sampling (USP <905> with 10-unit CUSUM). Key: blend uniformity RSD ≤5% at 10 locations (top/middle/bottom, thief sampling).

  

**Q6–Q10 (Brief):**

- Q6: "What is the role of pre-compression in a rotary tablet press?" → Deaerates granules, reduces capping/lamination, critical for high-speed presses (>60 RPM)

- Q7: "How does particle size of API affect dissolution and content uniformity?" → Smaller PSD → faster dissolution but poorer flow; needs balance

- Q8: "What is the significance of f1 and f2 values in dissolution?" → f1 (difference factor) ≤15, f2 (similarity factor) ≥50 for profile comparison

- Q9: "Describe USP dissolution Apparatus I vs II" → App I (basket, 100 RPM) for capsules/floating; App II (paddle, 50–75 RPM) for tablets

- Q10: "What is content uniformity vs weight variation? When do you use each?" → CU (USP <905>) for all; WV for >25mg dose and >25% of tablet weight

## 1.2 TABLETS — Extended Release (ER/SR)

  

### A) Complete Theory

  

**Extended Release (ER)** tablets release drug over a prolonged period (typically 8–24 hours) to reduce dosing frequency, minimize side effects, and maintain therapeutic plasma levels. Synonyms: SR (sustained release), CR (controlled release), XR/XL (extended release).

  

**Release Mechanisms:**

- **Matrix systems (monolithic):** Drug is dispersed in a polymer matrix; release occurs by diffusion through the matrix and/or erosion of the matrix. Most common in industry.

  - **Hydrophilic matrix:** HPMC (Methocel K100M, K15M) swells to form gel layer → drug diffuses through gel → outer gel erodes. *Example: Glucophage XR (Metformin)*

  - **Hydrophobic matrix:** Ethylcellulose, waxes (Compritol, Precirol). Drug diffuses through insoluble matrix. Erosion-independent. *Example: Certain Theophylline ER*

  - **Lipid matrix:** Glyceryl behenate (Compritol 888 ATO), stearic acid

- **Reservoir systems (membrane-controlled):** Drug core coated with a rate-controlling membrane (ethylcellulose, Eudragit). Release through membrane diffusion.

  - *Example: Adalat OROS (Nifedipine) — osmotic push-pull*

- **Osmotic systems (OROS):** Osmotic pressure pushes drug through laser-drilled orifice. Zero-order release. *Example: Procardia XL (Nifedipine), Concerta (Methylphenidate)*

  

**Release Kinetics Models:**

| Model                | Equation                   | When It Applies                                                                                 |
| :------------------- | :------------------------- | :---------------------------------------------------------------------------------------------- |
| **Zero-order**       | Q = Q₀ + K₀t               | Osmotic systems; ideal ER target                                                                |
| **First-order**      | log Q = log Q₀ - K₁t/2.303 | Conventional release; rate ∝ remaining drug                                                     |
| **Higuchi**          | Q = Kₕ√t                   | Drug release from matrix by diffusion                                                           |
| **Korsmeyer-Peppas** | Q = Kₖₚtⁿ                  | Characterizes mechanism: n=0.45 (Fickian), n=0.45-0.89 (anomalous), n=0.89 (Case II/zero-order) |
| **Hixson-Crowell**   | Q₀^(1/3) - Qt^(1/3) = Kₛt  | Erosion-dominated release                                                                       |

  

### B) Critical Formulation Parameters

  

| Parameter                          | Target                                                      | Why Critical                                              |
| :--------------------------------- | :---------------------------------------------------------- | :-------------------------------------------------------- |
| **Polymer type & viscosity grade** | HPMC K100M (100,000 cps) for 24h; K15M for 12h              | Controls gel strength and erosion rate                    |
| **Polymer concentration**          | 20–40% w/w for hydrophilic matrix                           | Below 20% → dose dumping; above 40% → incomplete release  |
| **Drug:polymer ratio**             | Typically 1:0.5 to 1:2                                      | Determines release duration                               |
| **Particle size of polymer**       | Fine particle → faster gel; coarse → slower initial release | Affects gel formation speed                               |
| **Drug solubility**                | BCS Class I/III preferred                                   | Poorly soluble drugs may need lipid matrix or solubilizer |
| **Tablet geometry**                | Surface area:volume ratio                                   | Affects release rate                                      |
| **Compression force**              | Formulation-dependent                                       | Affects porosity → water penetration → release rate       |

### F) Troubleshooting ER Tablets

| Problem                                             | Cause                                                 | Solution                                                                                     |
| :-------------------------------------------------- | :---------------------------------------------------- | :------------------------------------------------------------------------------------------- |
| **Dose dumping** (burst release)                    | Insufficient polymer; cracked tablet; alcohol-induced | Increase HPMC%; ensure tablet integrity; test alcohol-induced dose dumping (FDA requirement) |

## 1.3 TABLETS — Orally Disintegrating Tablets (ODT)

  

### Key Facts

- Disintegrate on tongue in **<30 seconds** (FDA) or **<3 minutes** (EMA)

- For pediatric, geriatric, psychiatric patients; patients with dysphagia

- **No water needed** for administration

- *Examples: Zyprexa Zydis (Olanzapine), Claritin RediTabs (Loratadine), Zofran ODT (Ondansetron)*

  

### Technologies

  

| Technology                | Method                                         | Example Product |
| :------------------------ | :--------------------------------------------- | :-------------- |
| **Zydis** (freeze-drying) | Lyophilized wafer in blister                   | Zyprexa Zydis   |
| **Durasolv**              | Conventional DC with high superdisintegrant    | —               |
| **Orasolv**               | Effervescent + taste-masking                   | —               |
| **FlashTab**              | Microgranules + superdisintegrant in DC        | —               |
| **Pharmaburst**           | Co-processed excipient for DC                  | Many generics   |
| **Direct Compression**    | High crospovidone (5–8%) + mannitol (DC grade) | Most generics   |

  

### Critical Excipients for ODT

  

| Excipient                                 | Role                                         | Concentration             |
| :---------------------------------------- | :------------------------------------------- | :------------------------ |
| **Mannitol (Pearlitol SD 200)**           | Sugar alcohol filler; pleasant cooling taste | 50–80%                    |
| **Crospovidone (Polyplasdone XL-10)**     | Fast disintegrant; porous                    | 5–10%                     |
| **Croscarmellose sodium**                 | Wicking + swelling disintegrant              | 3–6%                      |
| **Magnesium stearate**                    | Lubricant                                    | 0.5%                      |
| **Aspartame / Sucralose**                 | Sweetener                                    | 0.5–2%                    |
| **Flavors**                               | Taste masking                                | 0.5–2%                    |
| **Taste-masking coating** (Eudragit E PO) | Polymer coating on drug particles            | Depends on API bitterness |

  

### Interview Q: "How do you formulate an ODT for a bitter drug?"

> **Model Answer:** Three-pronged approach: (1) **Taste masking** — coat API particles with Eudragit E PO (dissolves at pH <5 in stomach, insoluble at salivary pH 6.8) or ethylcellulose using fluid bed coating. (2) **Cyclodextrin complexation** — HP-β-CD can complex with bitter APIs to block taste receptor interaction. (3) **Ion exchange resin** — Amberlite IRP-69 for drugs like dextromethorphan. Then formulate the taste-masked particles in a DC blend with mannitol (60–70%) + crospovidone (5–8%) targeting <30 sec disintegration. Test using e-tongue or trained human taste panel.

## 1.4 BILAYER TABLETS

  

**Purpose:** Combine two drugs in one tablet, or combine IR + ER layers for a drug, or separate incompatible drugs.

  

**Examples:** Glucovance (Metformin + Glyburide), Caduet (Amlodipine + Atorvastatin)

  

### Key Challenges

  

| Challenge                         | Solution                                                                                     |
| :-------------------------------- | :------------------------------------------------------------------------------------------- |
| **Layer adhesion (delamination)** | Optimize first-layer tamping force; ensure compatible excipients; roughen interface          |
| **Cross-contamination**           | Clean die table between layers; optimize powder flow                                         |
| **Weight control**                | Requires bilayer tablet press (e.g., Korsch XL 400, Fette 2090i) with separate fill stations |
| **Dissolution interference**      | Excipients from one layer affecting other layer's release                                    |

  

### Interview Q: "What causes delamination in bilayer tablets? How do you fix it?"

> **Model Answer:** Delamination occurs when the bond between layers is weak. Causes: (1) Excessive first-layer compression force (creates a smooth, non-porous surface). Solution: Use low tamping force (1–3 kN) for the first layer — just enough to form a coherent layer. (2) Incompatible excipients creating a weak interface. Solution: Use a common binder in both layers. (3) Excessive lubricant at the interface. Solution: Minimize Mg stearate in both layers. (4) Elastic recovery mismatch between layers. Solution: Reformulate to balance compaction behavior. Testing: Use side-crushing force test to measure inter-layer bonding strength.

## 1.5 CAPSULES

  

### Hard Gelatin Capsules (HGC)

| Parameter             | Details                                                                      |
| :-------------------- | :--------------------------------------------------------------------------- |
| **Shell composition** | Gelatin (bovine/porcine), water (13–16%), colorants, opacifier (TiO₂)        |
| **Sizes**             | 000 (largest, ~1.37 mL) → 5 (smallest, ~0.13 mL); Size 0 and 1 most common   |
| **Fill materials**    | Powders, pellets, granules, semi-solids, tablets (tablets-in-capsule)        |
| **Machine**           | Automatic capsule filler (Bosch GKF, MG2, IMA Zanasi)                        |
| **Advantages**        | Easy to swallow; taste masking; flexible fill combinations                   |
| **Limitations**       | Gelatin cross-linking (pellicle formation) with aldehydes; hygroscopic drugs |
| **Example**           | Omeprazole pellet-filled capsules (enteric-coated pellets in HGC)            |

  

### HPMC Capsules (Vcaps Plus, DRcaps)

- **Vegetarian** alternative (no animal-derived gelatin)

- **Lower moisture content** (4–6% vs 13–16% for gelatin) → better for hygroscopic/moisture-sensitive APIs

- **DRcaps:** Delayed-release HPMC capsules (resist acid, open in intestine) — no need for enteric coating of pellets

- *Growing rapidly in India and globally*

  

### Soft Gelatin Capsules (SGC)

  

| Parameter         | Details                                                      |
| :---------------- | :----------------------------------------------------------- |
| **Shell**         | Gelatin + plasticizer (glycerin/sorbitol) + water            |
| **Fill**          | Liquids, pastes, suspensions in oils or PEG                  |
| **Key use**       | Lipophilic drugs, SEDDS/SNEDDS, vitamins (A, D, E, K)        |
| **Manufacturing** | Rotary die process (seamless or die-formed)                  |
| **Example**       | Neoral (Cyclosporine SNEDDS in SGC), Accutane (Isotretinoin) |
| **Challenge**     | Drug migration into shell; shell-fill compatibility          |

## 1.6 PELLETS & MULTIPARTICULATES

  

### Theory

- **Pellets** = small spherical particles (0.5–2.0 mm) containing drug, prepared by extrusion-spheronization or layering

- **Multiparticulates** = multiple-unit dosage form → each pellet is an independent delivery unit

- **Advantage over monolithic:** More uniform GI transit; less risk of dose dumping; flexible release profiles

- *Example: Omeprazole DR capsules (enteric-coated pellets), Toprol-XL (pellets in tablet)*

  

### Manufacturing Methods

  

| Method                       | Process                                                                                     | Use                          |
| :--------------------------- | :------------------------------------------------------------------------------------------ | :--------------------------- |
| **Extrusion-Spheronization** | Wet mass → extruder (screen 0.7–1.0mm) → spheronizer (2–5 min, 700–1200 RPM) → drying (FBD) | Drug-loaded pellets          |
| **Drug Layering**            | Sugar/MCC spheres (Nonpareils) → coat with drug + binder solution in Wurster coater         | Low-dose, high-potency drugs |
| **Wurster Coating**          | Bottom-spray fluid bed coating → apply functional coating (enteric, SR)                     | Functional film coating      |

  

### Coating Types for Pellets

  

| Coating               | Polymer                                                         | Purpose                        | Example                 |
| :-------------------- | :-------------------------------------------------------------- | :----------------------------- | :---------------------- |
| **Enteric**           | Eudragit L30D-55 (dissolves at pH >5.5), HPMC phthalate         | Acid protection                | Omeprazole pellets      |
| **Sustained release** | Eudragit RS/RL 30D (pH-independent), Ethylcellulose (Surelease) | Extended release               | Diltiazem ER capsules   |

## 2.2 SUSPENSIONS

  

**Definition:** Coarse dispersion of insoluble drug particles (1–50 µm) in a liquid vehicle. *Example: Amoxicillin oral suspension*

  

### Key Concepts

  

- **Deflocculated:** Particles settle slowly as a hard cake (difficult to redisperse) → NOT desirable

- **Flocculated:** Particles form loose aggregates (flocs) → settle fast but redisperse easily → DESIRABLE

- **Structured vehicle:** Suspending agent creates a network that holds particles → prevents settling

  

### Critical Excipients

  
| Excipient                                | Role                                  | Concentration  |
| :--------------------------------------- | :------------------------------------ | :------------- |
| **Xanthan gum**                          | Suspending/thickening (pseudoplastic) | 0.1–0.5%       |
| **CMC-Na (Sodium CMC)**                  | Suspending agent                      | 0.5–2.0%       |
| **Veegum (Magnesium aluminum silicate)** | Suspending, thixotropic               | 0.5–2.0%       |
| **Tween 80 (Polysorbate 80)**            | Wetting agent                         | 0.1–0.5%       |
| **Methylparaben + Propylparaben**        | Preservative system                   | 0.18% + 0.02%  |
| **Citric acid + Sodium citrate**         | Buffer                                | pH 4–6 typical |
  

### Interview Q: "Why do we prefer flocculated suspensions?"

> Flocculated suspensions have loosely bonded particle aggregates that settle rapidly but form a soft, easily redispersible sediment. Deflocculated suspensions settle slowly but form a hard, compact cake that is nearly impossible to redisperse — this is dangerous because the patient gets a variable dose. We control flocculation using electrolytes (NaCl), surfactants, or oppositely charged polymers to create controlled, reversible aggregation.

## 3.2 Lyophilized Products

  

### Freeze-Drying Cycle Design

  

```

Step 1: FREEZING (-40 to -50°C, 2-4 hours)

   → Product freezes; ice crystals form; solutes concentrate between ice crystals

   → Must freeze below Tg' (glass transition of maximally freeze-concentrated solute)

   → Annealing step (optional): hold at -10 to -20°C to grow larger ice crystals (faster sublimation)

       ↓

Step 2: PRIMARY DRYING (Shelf temp: -20 to -10°C; Chamber pressure: 50-200 mTorr)

   → Ice sublimes (solid → vapor) under vacuum

   → Longest step (12-48+ hours)

   → Product temp must stay below Tc (collapse temperature)

   → Monitor by Pirani gauge, thermocouple, or comparative pressure measurement

       ↓

Step 3: SECONDARY DRYING (Shelf temp: +25 to +40°C; same vacuum)

   → Removes bound/unfrozen water from amorphous matrix

   → Target: residual moisture <1-3% for stability

   → Duration: 3-8 hours

       ↓

Step 4: STOPPERING (under vacuum or nitrogen backfill)

   → Vials stoppered inside lyophilizer

   → Crimped with aluminum seals

```

  

### Key Lyo Excipients

  
| Excipient                        | Role                                                    | Typical Conc. |
| :------------------------------- | :------------------------------------------------------ | :------------ |
| **Sucrose**                      | Lyoprotectant (stabilizes protein by water replacement) | 2–10%         |
| **Trehalose**                    | Lyoprotectant (higher Tg than sucrose)                  | 2–10%         |
| **Mannitol**                     | Bulking agent (forms crystalline cake)                  | 2–5%          |
| **Glycine**                      | Bulking agent + tonicity                                | 1–3%          |
| **Polysorbate 80**               | Surfactant (prevents protein aggregation)               | 0.01–0.05%    |
| **Histidine / Phosphate buffer** | pH control                                              | 10–50 mM      |

  

### Interview Q: "Why do we use sucrose or trehalose in lyophilized biologics?"

> **Model Answer:** During freeze-drying, removal of the hydration shell from proteins can cause unfolding and aggregation. Sucrose and trehalose act as **lyoprotectants** via two mechanisms: (1) **Water replacement hypothesis** — sugar hydroxyl groups form hydrogen bonds with protein surface, replacing water molecules and maintaining native conformation. (2) **Vitrification** — they form an amorphous glass matrix (high Tg) that immobilizes protein molecules, preventing conformational changes during storage. Trehalose is often preferred over sucrose because it has a higher Tg (115°C vs 62°C for sucrose), allowing for higher secondary drying temperatures and better long-term stability. Key: the sugar:protein molar ratio should typically be >300:1 for adequate stabilization.

# 5. ADVANCED DRUG DELIVERY SYSTEMS (ADDS)

  

## 5.1 Nanoparticles

  

### Polymeric Nanoparticles

- **Size:** 10–1000 nm (typically 100–300 nm for drug delivery)

- **Polymers:** PLGA, PLA, PCL, chitosan, albumin

- **Methods:** Nanoprecipitation, emulsion-solvent evaporation, double emulsion (w/o/w)

- **Example:** **Abraxane** (nab-paclitaxel) — albumin-bound paclitaxel nanoparticles (~130 nm), eliminates need for Cremophor EL solvent

  

### Solid Lipid Nanoparticles (SLN)

- Solid lipid core (e.g., glyceryl monostearate, Compritol) stabilized by surfactant

- Size: 50–1000 nm

- Advantage: Biocompatible lipids; controlled release; can be lyophilized

  

## 5.2 Liposomes

  

| Type                    | Composition                       | Size       | Example                                 |
| :---------------------- | :-------------------------------- | :--------- | :-------------------------------------- |
| **Conventional**        | Phosphatidylcholine + cholesterol | 100–400 nm | Ambisome (Amphotericin B)               |
| **Stealth (PEGylated)** | + DSPE-PEG 2000 (5–10 mol%)       | 80–120 nm  | Doxil (PEGylated liposomal doxorubicin) |
| **Cationic**            | + DOTAP or DOTMA                  | 50–200 nm  | Gene delivery                           |
| **Thermosensitive**     | + DPPC (transition at ~41°C)      | 80–150 nm  | ThermoDox (in clinical trials)          |

  

### Interview Q: "What is the difference between liposomes and LNPs?"

> **Model Answer:** Both are lipid-based nanovesicles but differ fundamentally. **Liposomes** have a well-defined **lipid bilayer** enclosing an aqueous core — hydrophilic drugs go in the core, lipophilic drugs in the bilayer. **LNPs** (Lipid Nanoparticles) have a more **complex internal structure** — they are NOT simple bilayer vesicles. LNPs typically contain ionizable cationic lipids that form an inverted hexagonal phase encapsulating nucleic acid (mRNA/siRNA) in the core. Key compositional difference: LNPs use **ionizable lipids** (e.g., DLin-MC3-DMA, ALC-0315, SM-102) that are neutral at physiological pH (avoiding toxicity) but become positively charged at endosomal pH (~5) to facilitate endosomal escape. LNPs are the delivery platform for COVID-19 mRNA vaccines (Comirnaty, Spikevax).

### Manufacturing: Microfluidic Mixing

  

```

Ethanol phase (lipids dissolved in ethanol)

       ↓

       → T-junction or staggered herringbone mixer ←

       ↓

       ↓

```

## 5.4 PLGA-Based Microspheres

  

**PLGA (Poly(lactic-co-glycolic acid))** — FDA-approved biodegradable polymer

  

### Marketed Examples

  

| Product              | Drug               | Release Duration     | Indication                     |
| :------------------- | :----------------- | :------------------- | :----------------------------- |
| **Lupron Depot**     | Leuprolide acetate | 1, 3, 4, or 6 months | Prostate cancer, endometriosis |
| **Risperdal Consta** | Risperidone        | 2 weeks              | Schizophrenia                  |
| **Vivitrol**         | Naltrexone         | 1 month              | Alcohol/opioid dependence      |
| **Bydureon**         | Exenatide          | 1 week               | Type 2 diabetes                |

  

### PLGA Degradation

- **Mechanism:** Bulk hydrolysis of ester bonds → lactic acid + glycolic acid (Krebs cycle metabolites) → CO₂ + H₂O

- **Factors affecting degradation:** Lactide:glycolide ratio (50:50 degrades fastest; 85:15 slowest), molecular weight, end-cap (acid-terminated degrades faster than ester-terminated)

- **Drug release:** Triphasic — initial burst (surface drug) → lag phase (slow diffusion through intact polymer) → erosion phase (rapid release as polymer disintegrates)

  

## 5.5 Hydrogels

  

- **Definition:** 3D cross-linked polymer networks that swell in water but do not dissolve

- **Types:** Chemical (covalent cross-links, permanent) vs Physical (ionic, H-bonding, reversible)

- **Stimuli-responsive:** pH-sensitive (Eudragit), temperature-sensitive (PNIPAAm, LCST ~32°C), enzyme-responsive

- **Applications:** Wound dressings, ophthalmic inserts, injectable depots, tissue engineering

> **Interview Master Tip:** When asked "Which excipient would you use for X?", always give:

> 1. The specific excipient name and brand

> 2. The concentration range

> 3. WHY you chose it (mechanism)

> 4. What compatibility issue to check

> 5. What alternative you'd consider if #1 doesn't work

>

> Example: "For a disintegrant, I'd use **croscarmellose sodium (Ac-Di-Sol) at 3% intragranular + 2% extragranular**. It works by wicking and swelling. I'd verify compatibility with the API via DSC/accelerated stability. If CCS doesn't give adequate disintegration (<15 min), I'd switch to **crospovidone (Polyplasdone XL) at 5%** which uses a non-swelling wicking/elastic recovery mechanism."
