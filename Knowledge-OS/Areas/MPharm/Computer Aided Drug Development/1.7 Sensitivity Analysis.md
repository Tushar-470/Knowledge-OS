# Sensitivity Analysis (10 Marks)

## Introduction

Sensitivity Analysis (SA) is a technique used to determine how changes in model parameters affect the output of a model. In pharmaceutical research, it helps identify the most influential parameters affecting drug behavior, formulation performance, and pharmacokinetic predictions. It is an important tool for model validation, optimization, and regulatory submissions.

---

# Definition

Sensitivity Analysis is the study of how variations in input parameters influence the output of a mathematical or statistical model.

In simple words:

**"It measures how sensitive a model output is to changes in its input parameters."**

---

# Objectives of Sensitivity Analysis

1. Identify important model parameters.
    
2. Assess model robustness.
    
3. Improve parameter estimation.
    
4. Guide experimental design.
    
5. Support regulatory acceptance of models.
    

---

# Need for Sensitivity Analysis

- Pharmaceutical models contain many uncertain parameters.
    
- Small changes in parameters may significantly affect predictions.
    
- Helps determine which parameters require accurate measurement.
    
- Reduces uncertainty during drug development.
    

---

# Types of Sensitivity Analysis

## 1. Local Sensitivity Analysis

### Definition

Local sensitivity analysis evaluates the effect of a small change in one parameter while keeping all other parameters constant.

### Sensitivity Coefficient

Sᵢ = (∂y/∂θᵢ) × (θᵢ/y)

where:

- Sᵢ = Sensitivity coefficient
    
- y = Model output
    
- θᵢ = Parameter
    

### Interpretation

- S > 1 → Highly sensitive parameter
    
- S < 1 → Less sensitive parameter
    

### Advantages

- Simple
    
- Fast
    
- Easy to interpret
    

### Limitation

- Studies only one parameter at a time.
    

---

## 2. Global Sensitivity Analysis

### Definition

Global sensitivity analysis evaluates the effects of simultaneous variations in multiple parameters over their entire range.

### Advantages

- More realistic.
    
- Considers parameter interactions.
    
- Suitable for complex pharmaceutical models.
    

### Limitation

- Computationally intensive.
    

---

# Methods of Global Sensitivity Analysis

## 1. Morris Method

- One-At-a-Time (OAT) approach.
    
- Identifies influential parameters.
    
- Used for preliminary screening.
    

## 2. Sobol Method

- Variance-based approach.
    
- Quantifies contribution of each parameter to output variability.
    
- Considered the gold standard.
    

Sobol Index:

Sᵢ = Vᵢ / V(Y)

where:

- Vᵢ = Variance due to parameter i
    
- V(Y) = Total output variance
    

## 3. PRCC (Partial Rank Correlation Coefficient)

- Measures relationship between input and output variables.
    
- Commonly used in population pharmacokinetic studies.
    

## 4. Deterministic Sensitivity Analysis (DSA)

- Parameters are varied by ±10%, ±20%, or ±50%.
    
- Widely used in PBPK modeling.
    

---

# Applications in Pharmaceutical Research

## 1. Pharmacokinetic Modeling

- Identifies parameters affecting drug concentration.
    
- Determines influence of clearance and volume of distribution.
    

## 2. PBPK Modeling

- Evaluates physiological parameters affecting drug disposition.
    

## 3. Formulation Development

- Identifies critical formulation variables.
    

## 4. Drug–Drug Interaction Studies

- Evaluates factors influencing interaction magnitude.
    

## 5. Population Pharmacokinetics

- Determines important covariates such as age, weight, and renal function.
    

---

# Regulatory Importance

FDA PBPK guidance recommends sensitivity analysis.

If changing an uncertain parameter by ±50% changes the predicted Drug–Drug Interaction (DDI) ratio by more than 2-fold, the model is considered highly sensitive and additional experimental verification may be required.

# Conclusion

Sensitivity Analysis is an important tool in pharmaceutical modeling that evaluates the effect of parameter changes on model output. It helps identify critical variables, improve model robustness, optimize experimental design, and support regulatory decision-making. Local and Global Sensitivity Analysis are the two major approaches, with Sobol analysis being the most widely accepted global method.
